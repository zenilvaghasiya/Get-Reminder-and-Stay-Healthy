# Get-Reminder-and-Stay-Healthy

This Pillbox has unique mechanism which track and send data of user medication activity to firebase and it will shows in user's phone.and This application will give reminder to user before medication time and This pillbox has one unique featues that is like If user take medicine 10-15 minutes before actual reminder time then automatically pillbox will not give reminder to user. because  somtimes we are annoying with so many notifications.
